# Number Guessing Game

A simple and interactive command-line Number Guessing Game written in Python. The computer randomly selects a number between 1 and 100, and the player has to guess it using helpful hints.

## Description

This project is a great beginner-friendly Python script that demonstrates the use of core programming concepts such as:
* The `random` module for generating random numbers.
* `while` loops for continuous gameplay.
* `if-elif-else` conditional statements for logic and user feedback.
* Handling user input and type casting.
